## üéØ Testing Tips

1. **Start Simple**: Begin with basic questions like "What is this document about?"
2. **Get Specific**: Ask detailed questions about specific sections or concepts
3. **Compare**: Try asking the same question in different ways
4. **Deep Dive**: Ask follow-up questions based on previous answers
5. **Extract Info**: Request specific data, numbers, or findings
6. **Summarize**: Ask for summaries of different lengths (brief, detailed)

- "Extract all the important dates/numbers/statistics"
